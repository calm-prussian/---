# TTS自动循环 - 酒馆助手脚本

简单说就是：**让角色在 SillyTavern 里自己跟自己一直聊，AI 的回复还能用语音念出来。** 你设定好要发什么文字（比如"继续"），点一下开始，它就会自动发过去 → AI 回复 → 语音朗读 → 再发 → 再回复 → 再朗读……一直循环下去。设置了轮次上限就会自动停，中间想停也可以随时暂停或终止。

电脑、手机都能用。

## 功能一览

| 功能 | 说明 |
|---|---|
| 🔊 TTS 朗读 | AI 回复自动合成语音播放，支持 21 种音色 + 语速调节 |
| 📤 自动发送 | 播放时自动发送下一条文本，支持启动/暂停/终止 |
| 🔄 文本重发 | AI 回复缺指定标签时自动删除并重新生成，可配置次数和间隔 |
| ✂ 文本过滤 | 只读区间（提取）→ 跳过区间（移除），解决 LLM 格式不听话 |
| 📊 轮次控制 | 设置 N 轮后自动停止，还可无限循环 |
| 📋 日志面板 | 实时查看运行日志，支持复制和清空 |
| 📱 移动端适配 | 面板自适应手机屏幕 |

## 安装

1. 安装 [酒馆助手](https://github.com/n0vi028/JS-Slash-Runner) 扩展
2. 在酒馆助手中导入 `酒馆助手-TTS自动循环.json`
3. 点击「TTS自动循环」按钮 → 打开控制面板
4. 设置好参数，点「▶ 开始」即可

## 使用流程

```
点「▶ 开始」→ 自动发送文本 → AI 生成回复
  → 过滤文本 → TTS 合成 → 朗读播放
  → 播放时自动发送下一条 → ...
  → 达到轮次上限或手动暂停/终止
```

## 提示

- TTS 依赖 `edgetts.velure.codes`，需要网络
- 文本过滤在 TTS 合成前执行：**只读区间 → 跳过区间 → 去 HTML → 朗读**
- 如果打开面板 CSS 显示异常，在控制台执行 `document.getElementById('ta-styles').remove()` 后重新导入
