# 自动TTS循环任务

## 快速开始

1. 酒馆 → 扩展设置 → 小白X扩展 → 循环任务 → 新增任务
2. 名称：`自动TTS循环`，触发时机：`手动触发`，楼层间隔：`0`
3. ✅ 勾选"激活任务栏按钮"
4. 复制 `auto-tts-loop.js` 全部内容，粘贴到命令框
5. 保存

## 使用

点击输入框上方任务栏按钮 → 弹出设置面板：

| 按钮 | 作用 |
|------|------|
| **开始循环** | 保存配置并开始自动发送 |
| **停止循环** | 立即停止当前循环 |
| **关闭** | 关闭面板 |

设置自动保存到浏览器，刷新不丢失。

## 工作原理

```
AI回复 → 点击TTS按钮 → TTS开始播放 → 立即发下一轮
```

同一条消息的多个TTS段落不会重复触发。
