# 自动TTS循环任务 - 安装指南

## 安装步骤（重要！）

### 1. 打开循环任务
酒馆 → 底部"扩展设置" → "小白X扩展" → "循环任务" → "新增任务"

### 2. 填写任务信息
- 任务名称：`自动TTS循环`
- 触发时机：`插件初始化` 或 `手动触发`
- 楼层间隔：`0`
- 楼层类型：`全部楼层`
- ✅ 勾选"激活任务栏按钮"（可选）

### 3. 粘贴代码（关键！）
**方式A（推荐）：**
1. 打开 `auto-tts-loop.js` 文件
2. **复制整个文件内容**（Ctrl+A 全选，然后复制）
3. 粘贴到循环任务的"命令"框中
4. 保存

**方式B（手动输入）：**
1. 在命令框中**手动输入**：`<<taskjs>>`
2. 换行
3. 复制 `auto-tts-loop.js` 中 `<<taskjs>>` 和 `</taskjs>>` 之间的内容（不包含这两行）
4. 粘贴到命令框
5. 换行
6. **手动输入**：`</taskjs>>`
7. 保存

## 为什么报错？

错误信息：`[ParseError: Unknown command at position 2: "/<>"]`

**原因：** 循环任务命令框期望的是 **STScript** 代码。如果要执行 JavaScript，必须用 `<<taskjs>>` 和 `</taskjs>>` 标签包裹！

**解决：** 确保命令框中的内容是：
```
<<taskjs>>
（这里是JavaScript代码）
</taskjs>>
```

## 测试

1. 如果触发时机选了`插件初始化`：刷新页面即可自动启动
2. 如果选了`手动触发`：点击任务栏的按钮启动

## 查看日志

按F12打开浏览器控制台，应该看到：
```
[AutoTTS] 第1次发送
[AutoTTS] 第2次发送
...
```

## 停止

- 刷新页面（最简单）
- 或运行：`window._autoTtsCleanup?.()`（如果保存了cleanup函数）

## 配置修改

编辑 `auto-tts-loop.js`，修改：
```javascript
const TEXT = '你的文本';  // 要自动发送的文本
const MAX_LOOP = 0;        // 0=无限循环
```

修改后，重新复制粘贴到循环任务中。
